# 音效檔案目錄

此目錄包含遊戲實況頁面使用的音效檔案。

## 音效檔案列表

- `game-start.mp3` - 遊戲開始音效
- `countdown.mp3` - 倒數計時音效
- `time-up.mp3` - 時間結束音效
- `correct-answer.mp3` - 正確答案音效
- `leaderboard.mp3` - 排行榜顯示音效
- `vote.mp3` - 投票音效

## 音效檔案要求

- 格式：MP3
- 檔案大小：每個檔案不超過 100KB
- 採樣率：44.1kHz
- 比特率：128kbps
- 時長：1-3 秒

## 注意事項

1. 音效檔案應該簡短明確，不影響遊戲流程
2. 音量應該適中，不會過於突兀
3. 音效風格應該與婚禮活動的氛圍相符
4. 考慮在不同設備和環境下的播放效果

## 臨時佔位檔案

目前使用佔位檔案，實際部署時需要替換為正式音效檔案。
