# LINE Rich Menu 圖片資源

本目錄用於存放 LINE Rich Menu 的圖片資源。

## 圖片規格要求

所有 Rich Menu 圖片必須符合以下規格：

- **尺寸**：2500 x 1686 像素（寬 x 高）
- **格式**：PNG 或 JPEG
- **檔案大小**：建議不超過 1MB
- **色彩模式**：RGB

## Rich Menu 類型

### 1. 會場資訊分頁 (venue_info)

**檔案名稱**：`venue-info.png` 或 `venue-info.jpg`

**按鈕配置**（2x2）：
```
+------------------+------------------+
|  交通資訊        |  菜單           |
|  (0,0,1250,843)  |  (1250,0,1250,843)|
+------------------+------------------+
|  桌次            |  進入遊戲分頁   |
|  (0,843,1250,843) |  (1250,843,1250,843)|
+------------------+------------------+
```

**設計建議**：
- 使用清新的配色（藍色、綠色、紫色）
- 每個按鈕區域使用不同的背景色
- 添加圖標或圖示增加識別度
- 文字清晰易讀

### 2. 現場活動分頁 (activity)

**檔案名稱**：`activity.png` 或 `activity.jpg`

**按鈕配置**（2x2）：
```
+------------------+------------------+
|  照片上傳        |  祝福照片牆       |
|  (0,0,1250,843)  |  (1250,0,1250,843)|
+------------------+------------------+
|  快問快答          |  進入會場資訊分頁|
|  (0,843,1250,843) |  (1250,843,1250,843)|
+------------------+------------------+
```

**設計建議**：
- 使用活潑的配色（粉色、橙色、黃色）
- 每個按鈕區域使用不同的背景色
- 添加圖標或圖示增加識別度
- 文字清晰易讀

### 3. 未開放分頁 (unavailable)

**檔案名稱**：`unavailable.png` 或 `unavailable.jpg`

**按鈕配置**：無按鈕

**設計建議**：
- 使用灰色或單色調
- 顯示「尚未開放」或「功能未開放」文字
- 可以添加裝飾性圖案
- 保持簡潔

## 上傳步驟

1. 準備符合規格的圖片檔案
2. 將圖片檔案放到本目錄
3. 透過後台管理介面上傳圖片到 LINE Platform
4. 系統會自動驗證圖片尺寸並上傳

## 設計工具推薦

- **Figma**：專業設計工具，支援精確的像素控制
- **Canva**：線上設計工具，有許多模板可使用
- **Adobe Photoshop**：專業圖片編輯工具
- **GIMP**：免費開源圖片編輯工具

## 注意事項

- 確保圖片在高解析度螢幕上仍然清晰
- 避免使用過多的漸層或複雜效果
- 文字與背景對比度要足夠高
- 測試在不同裝置上的顯示效果
