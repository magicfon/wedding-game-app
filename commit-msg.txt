🎨 修復 QR Code 載入狀態在白色背景上的文字顏色問題

🐛 修復問題：
- QR Code 載入狀態的文字在白色背景上看不見
- 載入中的文字顏色與背景對比度不足

✨ 修復方案：
- 載入狀態文字：text-gray-600 → text-gray-700 和 text-gray-800
- QR Code 圖標：text-gray-400 → text-gray-500
- 載入文字：添加 font-semibold 增強可讀性
- QR Code 圖片：添加 object-contain p-4 確保完整顯示

🎯 修復內容：
- 載入狀態容器：深灰色文字在白色背景上清晰可見
- QR Code 圖標：適當的灰色層次
- 載入提示文字：更深的顏色和更粗的字體
- QR Code 圖片：添加內邊距和適當縮放

📱 用戶體驗：
- QR Code 載入狀態現在清晰可見
- 文字與白色背景形成良好對比
- 載入提示更加明顯和專業
- QR Code 顯示更加完整和美觀
